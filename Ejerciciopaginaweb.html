Los tres pilares de la web



Tim Berners-Leees considerado el padre de la Web porque desarrolló los tres elementos básico para el funcionamiento de la Web:


HTML HTTP URL

HTML

HTML, siglas de HyperText Markup Language, hace referencia al lenguaje de marcado para la elaboración de páginas web. Es un estandar que sirve de referencia para la elaboración de páginas web en sus diferentes versiones, define una estructura básica y un código (denominado código html)para la definición de contenido de una página web, como texto,imágenes,entre otro. EsUN ESTANDAR a cargo de la W3C, organización dedicada a la estandarización de casi todas las tecnologias ligadas a la web, sobre todo en lo referente a su escritura e interpretación. Es el lenguaje con el que define las paginas web.
